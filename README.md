\## Problem Statement

The objective of this project is to predict whether a bank customer will subscribe

to a term deposit based on demographic and campaign-related features.



\## Dataset Description

\- Dataset: Bank Marketing Dataset (UCI)

\- Rows: 41,188

\- Features: 20

\- Target: y (yes/no)
